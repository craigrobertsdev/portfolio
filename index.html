Projects

EBrief

EBrief is an intuitive desktop application designed to increase productivity for time-poor prosecutors who have large court-list to prepare for. By utilising a lightweight database that comes preinstalled on every Windows computer, once the prosecutor has downloaded the data for their court-list, they have access to all the information they need to conduct their business in court without being held back by slow network speeds or clunky record management systems. The application is designed to be able to integrate with any prosecuting agency's servers, and has no external dependencies that need to be managed separately.

Made using Blazor Hybrid, a prosecutor will either open a case list provided by the court, or manually enter file numbers, causing the application to download all necessary information and store it in a SQLite database. Once downloaded, the information is presented in a logical format designed for the best possible user experience. The application has a custom file type (*.court), which it exports court lists to to enable sharing and act as business continuity should other systems fail, or a prosecutor be suddenly unavailable.

TeachPlanner

TeachPlanner is a work-in-progress all-in-one digital lesson planner for South Australian classroom teachers to enable them to keep all of their lesson planning, resources and student assessments in one place. Teachers set the structure of their work week into a template that is provided as a base for all school weeks throughout the year. Lesson planning is then done on a lesson-by-lesson basis, linking stored resources, assessments and the South Australian curriculum together to provide a fully integrated experience.

This is a full-stack web application that is powered by Blazor Web Assembly on the front end and an ASP.NET Core Web API on the backend, with a MySQL database. Resources will be stored in Microsoft Azure to remove the need for teachers to keep track of physical storage media that is prone to being lost.

PhotoChute

PhotoChute is an easy to use platform to collaborate with your friends and share photos with one another. Simply create an account, a group and add friends to get started. Users can upload photos which are stored securely in the cloud and are able to be viewed and downloaded on request. While there is a relatively small limit to the number of photos that can be uploaded on a free account, users can pay for premium accounts to increase this, with payments securely handled by Stripe.

This application is developed using the MERN stack (MongoDB, React, Node and Express) and uses Microsoft Azure's cloud storage and serverless Function Apps to handle file storage and thumbnail creation. Queries are handled by GraphQL and Apollo, and the app is deployed to Heroku.
